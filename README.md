# PRODIGY_GA_03 – Text Generation using Markov Chains

## Introduction

As part of my Generative AI Internship at Prodigy InfoTech, I developed a Text Generation System using Markov Chains. This project explores the fundamentals of Natural Language Processing (NLP) by generating text based on probabilistic word transitions.

Unlike advanced transformer-based models, Markov Chains generate text by predicting the next word using the statistical patterns learned from the training text. This project provided valuable insights into the foundational concepts of language modeling and text generation.

---

## Project Objective

The objective of this project is to generate meaningful text using Markov Chain-based language modeling techniques.

The model analyzes patterns within the input text and generates new sentences based on learned word relationships and probabilities.
